# 💫 About Me:
Hi, I’m Ayush — a tech enthusiast passionate about building practical solutions using AI, IoT, and embedded systems. I enjoy turning ideas into working prototypes, from smart automation systems to real-time monitoring and control projects.

I actively work with ESP32, Raspberry Pi, computer vision, and web technologies, focusing on efficiency, scalability, and real-world impact. I believe in learning by building, experimenting, and improving through open-source collaboration.

Currently, I’m exploring AI-powered automation, edge computing, and system integration, while continuously strengthening my fundamentals in software and hardware development.
